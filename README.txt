  What is it?
  -----------
  The Average_degree program is writen in Python3.5. It calculates
  the average degree of a vertex in a Twitter hashtag graph over a 60-seconds 
  sliding window.


  Source code
  -----------------
  Average_degree.py
  TruncateNum.py
  TweetGraph.py


  User defined Classes
  --------------------
  The main function (Average_degree) calles the following:

  1.TruncateNum converts a floating number to a 2 decimal places number
  2.TweetGraph adds nodes and edges to a graph object and calculates average_degree
